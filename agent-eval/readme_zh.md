# AI Agent对工业领域支持的评估
[English](readme.md) · 中文

## Agent功能/特性支持
  - 非阻塞会话
  - 工具协议支持
    - HTTP
    - OpenTool
    - MCP
  - 工具策略支持
    - 工具集
    - 冲突策略
  - MultiAgent
  - Auto MultiAgent
  - API调用
  - SDK
  - 导入导出

#### 子表1-1
| 分数 | 指标 |
| -- | ----------------------------- |
| 10 | 完全异步，平均响应 < 500ms，99%以上请求成功 |
